The Problem

Application-layer DDoS attacks bypass traditional volumetric defenses because each bot sends traffic at normal, individual rates. Existing approaches either rely on static thresholds that generate elevated false-positive rates or route traffic through third-party scrubbing infrastructure, introducing latency and data-sovereignty concerns.

Architecture

SENTINEL detects coordinated behavior patterns instead of volume natively at the gateway boundary.

Key Innovations

Adaptive Baseline Profiling

EWMA model learns service-specific traffic patterns and adapts to diurnal cycles.

Persistence Tracking

Leaky accumulator filters transient spikes to distinguish attacks from organic surges.

Multi-Signal Correlation

Correlates independent behavioral signals to reduce false positives drastically by requiring multi-dimensional temporal synchronization.

Reproducibility

The entire Sentinel testbed and experimental simulation suite is open source.
